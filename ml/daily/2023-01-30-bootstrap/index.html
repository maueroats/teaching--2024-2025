ISL-05X Bootstrap and CV

R Programming Tools

Problem: R does not always extend functions from working on a single value to a sequence of values in the way that you want.

Example: suppose that you have a (not very impressive) function to measure standard deviation. It takes in a positive number c, generates 50 random numbers from a normal distribution with standard deviation one, multiplies them by all by c, then finds the resulting standard deviation.

sd_example_1 <- function(c) {
    xs = rnorm(50, sd=1)
    ys = c * xs
    sd(ys)
}

This function should produce a number with mean c.

Now you would like to modify this function so that you can run it on every element in a vector, so

sd_example_2(c(1,5))

produces approximately c(1,5) as an output.

Purrr

Yes, purrr is an unusual name.

  • ~ do_something(.x, whatever): Starting with ~ makes a function of one variable (.x). I use this to fill in other inputs when I only want one to change.
  • map_dbl(sequence, function): Applies the function to every item in the sequence.
